## ğŸš€ Getting Started

### Prerequisites
- Node.js 18+ 
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/rasike-a/rasike-portfolio.git

# Navigate to the project directory
cd rasike-portfolio
```

